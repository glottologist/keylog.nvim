INTRODUCTION                                      *keylog.nvim-introduction*

A Neovim plugin to log keystrokes in order to generate heatmaps


FEATURES                                              *keylog.nvim-features*

- Log keystrokes to a keystrokes.log data file

REQUIREMENTS                                      *keylog.nvim-requirements*

- Neovim >= v0.7.0 <https://github.com/neovim/neovim/releases/tag/v0.7.0>


INSTALLATION                                      *keylog.nvim-installation*

1. Install via your favorite package manager.

>lua
    -- lazy.nvim
    {
       "glottologist/keylog.nvim",
       opts = {}
    },
<

1. Setup the plugin in your `init.lua`. This step is not needed with lazy.nvim if `opts` is set as above.

>lua
    require("keylog").setup()
<

USAGE                                                    *keylog.nvim-usage*

keylog.nvim is enabled by default. You can change its state with the
following commands:

- `:Hardtime enable` enable keylog.nvim
- `:Hardtime disable` disable keylog.nvim
- `:Hardtime toggle` toggle keylog.nvim
- `:Hardtime clear` clear the keystrokes.log file


Your log file is at `~/.local/share/nvim/keystroke.log`.
